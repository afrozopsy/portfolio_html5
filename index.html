Work

oKoro

oKoro is a image editing app build using React-Native. You can apply filters and control the intensity of it. And export the edited image in your gallery. Technical Details: This project was made in React-Native. I used EXPO SDK for a lot of their tools. And react-gl was used to do the heavy lifting and actually blending the GLSL fragment shaders with the input image. It was a nice experience to see web technologies working right way in mobile with power of react-native.
Side Node: This project was made in March 2020. And it does requires a update as the GLSurface doesn't work on some cellphones. But anyways, It does work in my phone and may be it will on yours as well. Here is the link to download it. https://play.google.com/store/apps/details?id=com.okoropureand
